RedHat style for Vale

RedHat style for Vale

The RedHat style for Vale contains language rules suitable for documentation in Open Source projects and mandatory for Red Hat products. These language rules originates from:

  • Red Hat supplementary style guide for product documentation

  • Latest edition of the IBM Style Guide

  • Vale-compatible implementation of IBM’s Developer Editorial Style Guide

  • Custom Vale open-source dictionary for American English.

The Vale at Red Hat repository contains the RedHat style in the following directory:

.vale/styles/RedHat

To use the RedHat style in your project you can decide to be:

  • Using Vale CLI

  • Using vale in the IDE

  • Using vale GitHub Action

  • Adding vale configuration to a project
