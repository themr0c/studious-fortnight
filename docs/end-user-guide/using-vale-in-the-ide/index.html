Using vale in the IDE

Using vale in the IDE

Consider using Vale in your IDE when:

  • You want a direct feedback in your editor.

  • You use a supported IDE, such as Visual Studio Code.

Prerequisites

  • Vale CLI is installed and configured. See Using Vale CLI.

  • Visual Studio Code is installed.

Procedure

  1. In Visual Studio Code, install the Visual Studio Code extension for Vale using the Marketplace.

  2. Configure the extension to use Vale CLI rather than Vale Server in the extension settings (Preferences > Extensions > Vale > Use CLI).

    "vale.core.useCLI": true

  3. Restart Visual Studio Code.

Verification

  1. Open a content file causing vale alerts.

  2. Open the Problems view and search for vale alerts in it.
