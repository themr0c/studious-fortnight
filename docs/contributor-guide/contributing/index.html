Contributing

You can help by doing any of the following:

  • Use Vale with the style rules from the vale-at-red-hat repository, see Installing and configuring Vale

  • Report issues and request enhancements

  • Join the Slack channel, #vale-at-red-hat, in the CoreOS workspace.
