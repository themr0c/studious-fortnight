Vale at Red Hat repository

vale-at-red-hat repository

The vale-at-red-hat repository contains resources to validate language in documentation projects using Vale:

  • RedHat style for Vale in the .vale/styles/RedHat directory

  • Vale configuration files for usage in a local environment or by the GitHub Action.
