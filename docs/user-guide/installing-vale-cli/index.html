Installing and configuring Vale

To use vale CLI on any project in your local environment without further configuration in the projects, install the vale CLI and register the vale-at-red-hat Vale configuration as the default.

Prerequisites

  • The git tool is installed.

Procedure

  1. Install the Vale command-line tool on your workstation. See Installing Vale.

    On the Vale site, click the tab for your operating system.

  2. Get a copy of the vale-at-red-hat repository to your local environment.

    $ git clone git@github.com:redhat-documentation/vale-at-red-hat.git

  3. Register the vale-at-red-hat vale configuration as the default.

    $ cd vale-at-red-hat
$ ln -s "$PWD/.vale.ini" "$HOME/"
$ ln -s "$PWD/.vale" "$HOME/"
Verification

  1. Run the vale command on one of your content files.

    $ cd <project_directory>
$ vale <filename>

  2. Review the vale output and use it to update your content file.

  3. Re-run the same vale command to see the updated results.
